📝 Form Pendaftaran

⚠️ Pastikan file KRS bisa diakses publik (anyone with the link).

💻 Cara Upload KRS ke Google Drive (Laptop/PC)

  1. Buka Google Drive di browser kamu.
  2. Klik + Baru → Upload file dan pilih file KRS kamu (PDF disarankan).
  3. Setelah selesai diunggah, klik kanan file → pilih Dapatkan link.
  4. Ubah akses menjadi Siapa saja yang memiliki link dapat melihat.
  5. Salin link tersebut dan tempel ke kolom “Link Google Drive (KRS)” di atas.

📱 Cara Upload KRS lewat HP

  1. Buka aplikasi Google Drive di HP kamu (login pakai akun UG/Google).
  2. Tekan tombol “+” di kanan bawah → pilih Upload.
  3. Pilih file KRS kamu dari galeri/dokumen (format PDF disarankan).
  4. Setelah selesai diupload, tekan titik tiga (⋮) di file → pilih Kelola orang dan link.
  5. Ubah akses menjadi “Siapa saja yang memiliki link” dan pastikan “Dapat melihat”.
  6. Salin link-nya dan tempelkan ke kolom form pendaftaran di atas.

✨ Tips: Pastikan koneksi stabil agar file KRS terunggah sempurna sebelum disalin link-nya.
