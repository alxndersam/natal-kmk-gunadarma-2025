🧾 Tutorial Upload & Isi KRS

💻 Melalui Laptop

  1. Buka KRS kamu (PDF / screenshot).
  2. Upload ke Google Drive.
  3. Klik kanan → “Dapatkan tautan”.
  4. Ubah jadi “Siapa saja yang memiliki link dapat melihat”.
  5. Copy link-nya dan tempel di kolom Link KRS di atas.

📱 Melalui HP

  1. Buka aplikasi Google Drive → unggah file KRS kamu.
  2. Tekan titik tiga (⋮) di file → pilih “Kelola akses”.
  3. Ubah jadi “Siapa saja dengan tautan”.
  4. Salin link dan tempel ke kolom Link KRS.
